> PHP library to generate simple charts fast.

**PHPFastChart** is a simple and fast library that allows developers to create a variety of charts quickly and easily. It supports multiple chart types, customization options, and high performance rendering.

## Features
- **📊 Multiple Chart Types**: Bar, Line, Pie, Scatter, and Radar.
- **❗ Error Handling**: Clear, specific error messages with location information
- **🚀 High Performance**: Fast chart generation
- **🔒 Type Safe**: Full PHP 8.1+ type declarations and strict mode
- **⚡ Zero Dependencies**: Pure PHP 8.1+ implementation using only stdlib
- **🧪 Well Tested**: Unit tests with comprehensive coverage
- **📦 PSR-12 Compliant**: Modern PHP coding standards

## Requirements

- PHP 8.1 or higher

## Installation

```bash
composer require codryn/phpfastchart
```
